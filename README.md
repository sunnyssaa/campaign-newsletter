# campaign-newsletter
Non-Governmental Organization Newsletter Archive&amp;Insight

## What is this project?
This is a tool that collects NGO campaigns, analyzes trends, and generates insights.

## Why was it created?
It was created to identify trends and share insights with people working in the same industry.

## What features does it offer?
- Organizes trends in the social impact sector.
- Matches campaigns to relevant trends.
- Analyzes insights from campaigns.
- Suggests discussion questions to help colleagues share and explore insights together.

## How do you use it?
1. Review example campaigns.
2. Enter your content.
3. Check the generated output file.

## Future plans
We plan to add more campaigns and expand the feature set.
